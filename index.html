How to Enable Live Team Syncing

Follow these steps to connect this dashboard to a free database so your team's checkmarks and diagnostic notes update on all screens instantly:

  1. Go to the Firebase Console and log in with your Google account.
  2. Click Add project and name it (e.g. Nutron-Hub).
  3. Once loaded, go to Project Shortcuts / Build and click Firestore Database. Click Create database. Choose Start in test mode (this lets your team write updates without restrictions).
  4. Return to your project homepage, click the Web platform icon (</>) to register an app. Copy the const firebaseConfig = { ... } snippet.
  5. Open this index.html file in any code editor (like VS Code, Notepad++, or text editor).
  6. Locate line 430 in this file, find const YOUR_CUSTOM_FIREBASE_CONFIG, and paste your credentials there.
  7. Save, and upload back to your GitHub repository!
Note: Your database path runs securely on the compliant route /artifacts/{appId}/public/data/farms_state to protect cross-device transactions.
